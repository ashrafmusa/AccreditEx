## Part 4 — Pre-Built Project Template Specification

### How to Seed OHAS Projects in AccrediTex

Each of the 14 departments should become a **pre-built project template** in AccrediTex when an organization selects "Oman SMCS" as their accreditation program. The template seeds:

1. **Project Name** — e.g., "SMCS Chapter 6 — Operation Theatre"
2. **Standards Checklist** — each SMCS.XX standard becomes a checklist item
3. **Sub-measures** — each SMCS.XX.a/b/c becomes a task within that standard
4. **Conformity Levels** — Not Applicable / Not Compliant / Partially Compliant / Fully Compliant (matches SAT scoring)
5. **Evidence Fields** — each sub-measure has an evidence upload slot
6. **Comments** — free text per measure
7. **Compliance Score** — auto-calculated from SAT scoring model

### SAT Scoring Model
| Conformity Level | Score |
|-----------------|-------|
| Not Applicable (NA) | Excluded from denominator |
| Not Compliant (0) | 0 points |
| Partially Compliant (1) | 1 point |
| Fully Compliant (2) | 2 points |
| **Total Score** | Sum / (Max × 2) × 100% |
